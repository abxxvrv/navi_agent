NAVI

NAVI AGENT

AI 浪潮之下，人类最终的诺亚方舟。

Navi 在你的项目目录里读文件、改代码、执行命令、管理技能、保存会话——而所有数据，都留在你自己的机器上。

立即安装 访问 GitHub

WINDOWS · MACOS · LINUX —— 一行命令完成安装

为什么是"诺亚方舟"

大模型把越来越多的工作搬上云端，
会话、记忆与数据，也随之离开你的机器。

Navi 选择相反的方向：AI 的能力交付给你，数据的主权也留给你。这就是"本地优先"的全部含义。

数据自持

会话历史写入本机 SQLite，记忆、技能与配置集中在 ~/.navi 目录。可备份、可迁移、可删除，完全由你处置。

控制在你

写文件、跑命令，都要经过你设定的审批级别；运行中按 Ctrl+C 立即叫停。外部网关默认白名单关闭，授权才通行。

不绑定任何一方

五家模型供应商自由切换，也可以完全跑在本地模型上。换帆不换舟——方舟始终是你的。

核心特性

一艘装备完整的方舟

从会话档案到长期记忆，从多模型路由到即时通讯网关——出发需要的一切，都已在船上。

01

本地会话档案

会话历史写入 history.sqlite3，FTS5 全文索引。/search 检索往事、/fork 任意分叉、/compress 压缩长上下文——每段协作都有迹可循。

02

长期记忆与技能

任务结束后后台自动反思：沉淀长期记忆，维护并修补技能库。Navi 会记住你的项目与习惯，越用越顺手。

03

多模型路由

stepfun、Kimi、DeepSeek、LongCat、LM Studio 本地模型，/model 一键切换；恢复旧会话时自动沿用当时的模型。

04

审批与中断

strict / normal / open 三级审批，--yolo 全权委托；任何时刻 Ctrl+C 中断当前任务，已写入的会话历史保持完整。

05

内置工具箱

glob grep read_file write_file patch_file bash powershell vision_analyze search_session 等开箱即用，MCP 可接入更多工具。

06

CLI · Web · 即时通讯

交互式 CLI、本机 Web 控制台（navi web）、微信与 QQ 网关（白名单 fail-closed，!cancel 远程叫停）——在你顺手的地方找到它。

开始

三十秒启航

一行命令完成安装，三条命令进入工作状态。

WINDOWS · POWERSHELL

irm https://raw.githubusercontent.com/abxxvrv/navi_agent/gpt/install.ps1 | iex

MACOS · LINUX

curl -fsSL https://raw.githubusercontent.com/abxxvrv/navi_agent/gpt/install.sh | bash
navi init 初始化配置 → navi doctor 检查环境 → navi 进入交互模式

也可以 navi run "检查当前工作区的未提交修改" 单次执行任务，或 navi --continue 恢复上一次会话。
